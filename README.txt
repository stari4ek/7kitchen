Ok, this is just a quick overview of 7kitchen.

7kitchen is a bunch of scripts, created to help with 7hero custom rom developing.
Author: Michael Sotnikov
e-mail: stari4ek@gmail.com
7hero forum page (russian): http://4pda.ru/forum/index.php?showtopic=170135

Dependency. Should be accessable through the $PATH:
* apktool (http://code.google.com/p/android-apktool/)
* smali/backsmali (http://code.google.com/p/smali/)
* unyaffs (http://code.google.com/p/unyaffs/)
* mkbootimg (http://android.git.kernel.org/?p=platform/system/core.git;a=tree;f=mkbootimg;hb=HEAD)
* split_bootimg.pl (from William Enck)
* testsign.jar
